# Employee Attrition Analysis Project

Developed an end-to-end Employee Attrition Analysis solution using Python, Machine Learning, and Power BI to identify factors influencing employee turnover and predict attrition risk.

## 📌 Project Overview

Employee attrition is a critical challenge for organizations as it impacts productivity, recruitment costs, and business continuity. This project analyzes HR data to identify key factors affecting attrition and builds predictive models to identify employees who are likely to leave.

## 🎯 Objectives

- Analyze employee attrition patterns
- Identify factors influencing employee turnover
- Build machine learning models for attrition prediction
- Develop an interactive Power BI dashboard
- Generate actionable HR recommendations

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- SMOTE (Imbalanced-Learn)
- Power BI
- Excel

## 📊 Exploratory Data Analysis

Analyzed attrition based on:

- Salary Slab
- Overtime
- Business Travel
- Age
- Years at Company
- Stock Option Level

### Key Insights

- Employees working overtime had significantly higher attrition.
- Lower salary employees showed the highest turnover rate.
- Frequent business travel increased attrition risk.
- Younger employees were more likely to leave.
- Employees with fewer years at the company had higher attrition.
- Stock options improved employee retention.

## 🤖 Machine Learning Models

### Logistic Regression
- Accuracy: 82.3%

### Random Forest
- Accuracy: 83.3%

### Random Forest + SMOTE (Final Model)
- Accuracy: 86.7%
- Precision: 85%
- Recall: 32%
- F1-Score: 47%

## 💡 Recommendations

- Reduce excessive overtime workload.
- Improve compensation for lower salary employees.
- Provide flexible travel policies.
- Strengthen employee engagement during early tenure.
- Expand stock option and reward programs.
- Use predictive analytics for proactive retention planning.
